Google Site Verification

This file is used to verify ownership of compomint.dev for Google Search Console.

Please replace the meta tag verification method in the main HTML file with your actual verification code from Google Search Console.

Setup Instructions:

  1. Go to Google Search Console
  2. Add the property: https://compomint.dev
  3. Choose verification method (HTML tag recommended)
  4. Add the verification meta tag to your main index.html file
  5. Submit your sitemaps:
    • https://compomint.dev/sitemapindex.xml
    • https://compomint.dev/sitemap.xml
    • https://compomint.dev/sitemap-images.xml
